## 🛠 Hướng dẫn cài đặt & Chạy Docker

1. **Chạy Local:** Mở file index.html trực tiếp trên trình duyệt.

2. **Chạy Docker:**

  - Run container: docker run -d -p 8080:80 email-ai-app
